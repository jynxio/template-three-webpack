# template-threejs
## 简介

three.js 项目的模版。

## 运行

| 描述   | 命令             |
| ------ | ---------------- |
| 初始化 | `npm install`    |
| 运行   | `npm run start`  |
| 打包   | `npm run build`  |
| 部署   | `npm run deploy` |
